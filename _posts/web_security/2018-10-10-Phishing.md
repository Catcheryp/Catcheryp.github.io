## 0x0 JavaScript 键盘记录器

```html
<!--在 chrome 测试成功-->
<html>
<head>
  <meta charset="utf-8">
</head>
<body>
  <script>
    function keyDown(){
      var keycode = event.keyCode;
      var realkey = String.fromCharCode(event.keyCode);
      alert("按键码:" + keycode + "    字符:" + realkey);
    }
    document.onkeydown = keyDown;
  </script>
</body>
</html>
```
