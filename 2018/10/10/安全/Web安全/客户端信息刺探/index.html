0x01 JavaScript 实现端口扫描

JavaScript 可对用户本地网络中的主机进行端口扫描，从而确定可被利用的服务。

原理的关键点就是新建一个 Image 对象，将 src 设成目标网页 target:port，如果目标在使用这个端口那么就会发生 load 或者 error 事件。这就足以使脚本判断目标网址是否在使用查询端口了。
